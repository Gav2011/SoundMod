## What is SoundMod?

SoundMod is a soundboard app that lets you play audio clips system wide for use in Discord, games, streams, or any application. Press a button, play a sound. Simple as that.

> **Heads up:** Audio may sound low quality in Discord by default. Follow the setup steps below to fix this!

---

## Getting Started

### 1. Download & Install

Grab the latest version from **[modeminc.com/soundmod](http://modeminc.com/soundmod)** and run the installer.

### 2. Fix Audio Quality in Discord

By default, Discord compresses audio which makes soundboard output sound bad.

**User Settings > Voice & Video**

![Discord Audio Settings](https://raw.githubusercontent.com/Gav2011/SoundMod/refs/heads/master/Screenshot%202026-02-15%20014041.png)

## Features

- Play sounds directly into Discord voice channels
- Organize sounds into boards/categories
- Hotkey support for quick playback
- Simple, lightweight UI
